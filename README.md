# ProjetoTelaChamadaCentenario
Este repositório tem dentro de si o código do projeto sobre uma tela de chamada para o hospital centenário, além disso, serve para documentar o que foi feito e o progresso do mesmo.

O presente projeto tem como finalidade agilizar o atendimento e modernizar o pronto-socorro do hospital, visto que o projeto possui uma interface gráfica moderna e "limpa" para facilitar a visibilidade e
transmitir uma sensação de técnologia ao adentrar o setor. Também visa a resolução de problemas de comunicação entre os computadores do pronto-socorro pois o mesmo utiliza de uma interface antiga que precisa
do internet explorer, que já foi descontinuado. E por certos casos pode inviabilizar o andamento das chamadas.
